Map name		- map
Map location	- ./resources/map

Map format rules:
1. Minimum map size - 3x3.
2. The player spawns on the coordinates 1 1; The figure on these coordinates is ignoring.
3. 0 - is floor. The border of map can't be floor.
4. Symbols that are not [0, 1, .., 7] are invalid.

Map figures:
0 - floor
1 - red wall
2 - eagle
3 - stone
4 - axid stone
5 - purple stone
6 - wood
7 - figure with different sides
